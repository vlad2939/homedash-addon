HomeDash - README

HomeDash este o aplicație pentru analiza și vizualizarea cheltuielilor lunare ale casei (curent, gaze, apă, internet, gunoi, telefon, impozit).

Ce face aplicația

  • Centralizează date istorice pe ani și luni într-un format unitar.
  • Afișează un Dashboard configurabil cu KPI-uri și grafice.
  • Permite analiză dinamică după filtre (ani, lună, metrici).
  • Permite import manual/CSV și întreținere date din pagina Date.
  • Permite backup/restore complet (date + setări + configurații carduri).

Structură funcțională

  • Dashboard - vedere de ansamblu cu carduri configurabile.
  • Analiză - comparații și filtre pe intervale de timp.
  • Date - import CSV și adăugare/ștergere înregistrări.
  • Setări - configurare carduri, calcul impozit, backup/restore, documentație.

Persistență date

Datele aplicației sunt salvate local în browser prin localStorage.

Tehnologii folosite

  • HTML + JavaScript (fără build step).
  • Tailwind CSS (CDN) pentru layout și stilizare.
  • Chart.js pentru grafice interactive.
  • PapaParse pentru import și parsare CSV.
